Shopify Configuration

Here are our shopify configuration.

To connect to shopify you have to configure shop name, api key and password.

(Settings -> Shopify -> Connection)

To configure shopify you need input shop name, api key and password.

Import Customers

From Shopify

To import the customer , click on the "Import Customer". This will import the new customers and update the changes in the existing customer.

Import Products and Variants.

From Shopify

To import Products and Products Variants, click on the "Import Product and Variants" button. This will import the new products along with the new variants and update the existing products if any.

Import Sale Orders

From Shopify

To import the Sale Order, click on the "Import Sale Order" button. This will import all the new Sale Orders and update the existing Sale Order in case of any changes
